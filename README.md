# Excel task pane add-in using Vue

## See
```
https://docs.microsoft.com/en-us/office/dev/add-ins/quickstarts/excel-quickstart-vue
```

### Restore packages
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
